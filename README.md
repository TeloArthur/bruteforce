# bruteforce

Este repositório contém imagens de alguns testes feitos em ambiente Kali Linux para exploração de vulnerabilidades.
O protocolo FTP estava aberto e conseguimos ter acesso por meio de um ataque de força bruta.
Foram criadas duas wordlists uma com usuários e outra com possíveis senhas.
Após fazer o teste com as wordlists foram encontrados o usuário e senha.
Com isso conseguimos acessar dentro de FTP.
